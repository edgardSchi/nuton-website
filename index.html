Analysieren von Videos im naturwissenschaftlichen Unterricht

Nuton ist ein Programm, welches hilft Bewegungen in Videos zu analysieren. Es ist vorallem für den Schulunterricht entwickelt worden, kann jedoch auch anderweitig verwendet werden.

Plattformunabhängigkeit

Nuton wurde mit Java geschrieben und ist damit Plattformunabhängig. Es kann auf jedem Betriebssystem genutzt werden, welches von Java unterstützt wird.

Open Source

Nuton ist kostenlos und Open Source. Der Quellcode kann komplett eingesehen werden und Sie können bei der Entwicklung sogar helfen!
